Quiz API Links

1 - Add Quiz (Title, Description)
	POST : http://127.0.0.1:8000/api/quizzes
	*fields in form body
2 - Update Quiz
	PUT : http://127.0.0.1:8000/api/quizzes/{id}
	*fields in form body
3 - Delete Quiz
	DELETE : http://127.0.0.1:8000/api/quizzes/{id}
4 - Retrieve Quiz
	GET : http://127.0.0.1:8000/api/quizzes/{id}
5 - Retrieve All Quizzes
	GET : http://127.0.0.1:8000/api/quizzes
